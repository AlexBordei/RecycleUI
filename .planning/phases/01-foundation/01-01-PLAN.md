    After creation:
    - Verify Laravel 11.44.1+ is installed (CVE-2025-27515 fix requirement from research)

<task type="auto">
  <name>Task 2: Configure MySQL database</name>
  <files>.env, .env.example, config/database.php</files>
  <action>
    Configure MySQL in .env:
    - DB_CONNECTION=mysql
    - DB_HOST=127.0.0.1
    - DB_PORT=3306
    - DB_DATABASE=recycleui
    - DB_USERNAME=root (placeholder, user will update)
    - DB_PASSWORD= (empty placeholder, user will update)

    Update .env.example with same structure (without sensitive values).

    Test database connection with `php artisan migrate:status` (will show no migrations yet, but confirms connection).
    If connection fails, that's expected until user configures their MySQL credentials.
  </action>
  <verify>cat .env | grep DB_ shows MySQL configuration</verify>
  <done>MySQL configured in .env, ready for user to add credentials</done>
</task>
